# Gestor de Tareas

Aplicación web simple para gestionar tareas diarias.

## Funcionalidades

- Agregar tareas con fecha
- Marcar tareas como completadas
- Editar tareas
- Eliminar tareas
- Filtrar tareas por fecha
- Contador de tareas completadas y pendientes
- Persistencia de datos usando localStorage

## Tecnologías

- HTML
- CSS
- JavaScript

# Task Manager

Simple web application for managing daily tasks.

## Features

- Add tasks by date
- Mark tasks as completed
- Edit tasks
- Delete tasks
- Filter tasks by date
- Counter for completed and pending tasks
- Data persistence using localStorage
